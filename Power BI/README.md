# Dashboard – Mercado de Trabalho em TI
 Este dashboard foi desenvolvido para analisar o mercado de trabalho na área de TI no Brasil. 
 Ele permite visualizar salários, quantidade de vagas, habilidades mais requisitadas, distribuição geográfica e modalidade de trabalho

### 📊 KPIs principais
- Maior salário
- Menor salário
- Salário médio
- Total de vagas

### 📈 Gráficos e análises
- Mapa de distribuição das vagas, onde mostra as cidades com vagas ativas nos estados de SP, MG, RJ e ES.
- Gráfico de barras mostrando as competências mais demandadas: Git, SQL, Python, AWS, Java, GitHub, HTML/CSS/JS e Azure.
- Gráfico de rosca que mostra a distribuição por modalidade de trabalho
- gráfico de barras indicando a quantidade de vagas por estado.

### 💡Insights obtidos
- Cargos e salários: Engenheiros e cientistas de dados são os cargos mais bem pagos.
- Habilidades mais requisitadas: Git, SQL e Python lideram a demanda, destacando competências técnicas e de programação.
- Modalidade de trabalho: A maioria das vagas é remota, mostrando uma tendência de flexibilidade no mercado.
- Distribuição geográfica: São Paulo concentra a maior parte das oportunidades, seguido por Minas Gerais, Rio de Janeiro e Espírito Santo.
